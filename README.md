# fullstack-mern-email-survey-app

**_View this project:_** [https://fullstack-mern-email-survey-app.onrender.com/](https://fullstack-mern-email-survey-app.onrender.com/)

## Description

A fullstack application build with NodeJS, Express, React, Redux, and MongoDB. User login implemented with a Google OAuth authentication flow (Google+ API), facilitated by Passport JS. (CD/CI) Deployment on Render.

(Notes: remote MongoDB instance, Mongo model class, Mongo model instance, queries, integration with Mongoose library, integration of database with Oauth flow; configuration of sessions with Cookie Session NPM package/library, encryption and automatic expiration of cookie(s); serialization and deserialization of user and login/logout functionalities with Passport; attention to developer vs production environments (separate databases and deployments), environment variables; Concurrently package to efficiently run multiple servers (frontend and backend);...)
